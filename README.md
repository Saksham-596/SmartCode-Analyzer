🚀 SmartCode-Analyzer: AI-Powered Code Execution & Complexity EngineSmartCode-Analyzer is a full-stack platform that provides developers with a secure environment to execute code and receive instant, AI-driven feedback on computational efficiency. It combines a custom Machine Learning pipeline with LLM integration to bridge the gap between "writing code" and "optimizing code."🌟 Key FeaturesMulti-Language Execution: Instant execution for C++, Python, and Java via a secure JDoodle API bridge.Custom ML Classifier: A Random Forest model trained on thousands of C++ source files to predict Big O complexity.LLM Reflection: Fallback integration with Google Gemini API for high-accuracy logic analysis.Professional Editor: A sleek, dark-themed UI powered by the Monaco Editor (the engine behind VS Code).Dynamic Complexity UI: Real-time complexity badges that update based on structural code analysis.🏗️ Technical ArchitectureThe project is structured as a Monorepo to separate concerns between the UI and the high-compute ML engine.1. Frontend (/sandbox)Framework: Next.js 15 (App Router)State Management: React Hooks (useState, useEffect)Deployment: Vercel (Edge Network)2. ML Engine (/ml-engine)Framework: FastAPI (Python)Model: Random Forest Classifier (10-class)Data Source: Thousands of C++ files scraped from GitHub and competitive programming platforms.Deployment: Render🧠 The Machine Learning PipelineUnlike standard static analyzers, SmartCode uses a custom-built data pipeline to "understand" code structure.Data EngineeringCollection: I scraped thousands of C++ source files from GitHub, labeling them into 10 distinct complexity classes ($O(1)$, $O(n)$, $O(n^2)$, $O(\log n)$, etc.).Preprocessing: Used NLTK and Regex to tokenize code, stripping boilerplate and comments while preserving structural keywords.Model TrainingAlgorithm: Random Forest Classifier.Feature Extraction: Engineered features based on keyword frequency (e.g., for, while, if) and N-gram analysis to detect nested loop patterns.Performance: Achieved 55% accuracy across 10 classes—significantly outperforming the 10% random baseline and successfully identifying the "structural signature" of complex algorithms.🛠️ Challenges OvercomeMonorepo Deployment: Successfully configured Vercel's Root Directory and Framework Presets to handle a nested Next.js application within a larger project structure.Environment Security: Implemented secure Server-Side Environment Variables to protect JDoodle API credentials from client-side exposure.Linux Case-Sensitivity: Resolved Git-to-Vercel casing mismatches (Mac's case-insensitivity vs. Linux's strictness) that initially caused production 404 errors.Execution Sandboxing: Pivoted from a local Docker-based engine to a cloud-based execution API to ensure 100% uptime in production environments.
